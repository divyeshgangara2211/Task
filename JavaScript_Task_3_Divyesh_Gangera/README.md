# Task 3 - Table Pagination

## Overview
Client-side pagination for a dummy data table with Previous/Next buttons and numbered pages including ellipsis. Built with HTML, CSS, and vanilla JavaScript.

## Features
- Previous and Next buttons
- Numbered page buttons with three middle page buttons around the current page
- Ellipsis when page ranges are skipped
- Dummy users table (id, name, email, role)
- Rows per page selector (5 or 10)
- Keyboard navigation (Left/Right arrows)
- Responsive and accessible UI

## How to Run
1. Open `index.html` in any modern browser.
2. Use pagination controls below the table.
3. Change “Rows per page” to see different page sizes.
4. Use Left/Right arrow keys to move between pages.

## Files
- `index.html` – Markup for table and pagination
- `styles.css` – Styling for table and pagination component
- `script.js` – Pagination logic and dummy data
